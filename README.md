# 🌿 Flora Group — Ландшафтный дизайн и озеленение

Современный веб-сайт для компании по ландшафтному дизайну, благоустройству территорий и продаже комнатных растений.

## 🛠 Технологический стек

* **Framework:** [Next.js](https://nextjs.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Components:** [shadcn/ui](https://ui.shadcn.com/)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Language:** TypeScript / JavaScript

## 🚀 Быстрый старт

1.  **Клонируйте репозиторий:**
    ```bash
    git clone [https://github.com/DeliciousM1lk/flora_group](https://github.com/DeliciousM1lk/flora_group)
    ```

2.  **Установите зависимости:**
    ```bash
    npm install
    # или
    yarn install
    ```

3.  **Запустите сервер для разработки:**
    ```bash
    npm run dev
    ```
    Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## 📁 Структура проекта

* `/components` — переиспользуемые UI компоненты (Header, Footer, Button и т.д.).
* `/app` — основные страницы и роутинг сайта.
* `/public` — статические файлы (логотипы, изображения растений).
