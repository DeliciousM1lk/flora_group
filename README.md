# Flora Group — Unified Landscaping Platform

Масштабная экосистема для группы компаний **Flora Group** (Казахстан), объединяющая пять бизнес-направлений в единую цифровую платформу. Проект включает в себя каталог питомника, систему управления ландшафтными проектами и сервис абонентского обслуживания.

## 🏗 Архитектура проекта

Проект построен по модели **Decoupled Architecture**:
* **Frontend**: Высокопроизводительное SPA/SSR приложение на Next.js с упором на SEO и анимации.
* **Backend**: RESTful API на Django, обеспечивающее управление контентом пяти подразделений (Питомник, Ландшафт, Фитодизайн и др.).

## 🛠 Технологический стек

### Frontend
- **Next.js 15 (App Router)**: Маршрутизация и серверный рендеринг.
- **TypeScript**: Строгая типизация всей логики.
- **Tailwind CSS + Shadcn UI**: Современный адаптивный интерфейс с кастомной дизайн-системой (Oklch цвета).
- **Framer Motion**: Продвинутые анимации скролла и переходов.
- **Lucide React**: Библиотека иконок.

### Backend
- **Django 6.0**: Мощная админ-панель и ORM.
- **Django REST Framework (DRF)**: Гибкое API для мобильных и веб-клиентов.
- **PostgreSQL**: Надежное хранилище данных.
- **Pillow**: Автоматическая обработка и оптимизация изображений растений и проектов.

## 🌟 Ключевые возможности

* **Унифицированный каталог**: Разделение растений по категориям (хвойные, лиственные, цветы, комнатные).
* **Портфолио проектов**: Динамическая фильтрация выполненных работ по категориям и слагам.
* **Сервисные планы**: Реализована логика выбора тарифов на уход (Базовый, Стандарт, Премиум).
* **Интерактивность**: Интеграция с WhatsApp, анимированные CTA-секции и адаптивность под мобильные устройства (use-mobile hooks).
